# CareerLog 에이전트 사용법

> 기준일: 2026.03
> 모든 명령어는 `CareerLog/` 루트 폴더에서 실행

---

## 빠른 시작

```powershell
# 전체 파이프라인 한번에 (처음엔 이걸로 시작)
py agents/orchestrator.py
```

---

## 에이전트 목록

### 🎯 orchestrator.py — 전체 파이프라인 조율자

모든 에이전트를 순서대로 실행해주는 메인 진입점. 처음엔 이걸로만 써도 됨.

```powershell
py agents/orchestrator.py
```

**실행 순서:**
1. 기획서 구체화 (선택 — 대화형)
2. **기획서 자동 보완 (자동)**
3. 정부과제 조사 (자동)
4. 과제별 기획서 피팅 (선택한 과제들)
5. 심사위원 검토 (선택)
6. 예산 계획 (선택)

---

### 01. 기획서 구체화 에이전트

**역할:** 기획서에서 정부과제 심사에 필요하지만 빠진 항목을 대화로 채워줌
(시장 규모, 팀 역량, 재무 계획, 기술 차별성 등)

```powershell
py agents/01_planner.py
```

**대화 중 사용 가능한 명령어:**

| 명령어 | 설명 |
|--------|------|
| `/분석` | 현재 기획서에서 보완 필요한 항목 리스트 출력 |
| `/저장` | 대화로 추가된 내용을 기획서.md에 반영하고 버전 파일 생성 |
| `/종료` | 에이전트 종료 |

**출력:** `agents/outputs/기획서_vYYYYMMDD_HHMM.md`

---

### 01b. 기획서 자동 보완 에이전트

**역할:** 01_planner가 찾아낸 빈 항목들을 대화 없이 자동으로 채워서 기획서.md를 직접 업데이트함. 정부과제 심사에 필요한 7개 핵심 섹션 기준으로 빠진 것만 골라서 생성.

```powershell
# 기본 실행 (기획서.md 직접 수정)
py agents/01b_refiner.py

# 수정 없이 결과만 미리 보기
py agents/01b_refiner.py --dry-run
```

**자동 생성 대상 섹션:**

| 섹션 | 설명 |
|------|------|
| 창업자/팀 소개 및 역량 | 창업자 이력, 팀 구성, 향후 채용 계획 |
| 시장 규모 (TAM/SAM/SOM) | 국내 HR Tech 시장 규모, 유료 타겟 추정 |
| 기술 개발 상세 계획 | 단계별 개발 계획, 기술 차별성 |
| 재무 계획 (3년 추정) | 연도별 매출/비용/BEP |
| 사업화 전략 및 채널 | CAC, LTV, 파트너십 전략 |
| 사회적 가치 및 고용 효과 | 고용 창출, 노동시장 기여 |
| 지식재산권/기술 차별성 | 특허 계획, IP 전략 |

> 이미 있는 섹션은 건드리지 않고 빠진 것만 추가함

**출력:** `기획서.md` 직접 업데이트 + `agents/outputs/기획서_보완_YYYYMMDD.md` 백업

---

### 02. 정부과제 조사 에이전트

**역할:** 현재 신청 가능한 정부 지원사업 전수 조사. CareerLog 프로필에 맞는 과제를 찾아서 적합도 분석까지 해줌

```powershell
py agents/02_researcher.py
```

**조사 대상 기관:** 창업진흥원, TIPS, NIPA, 고용노동부, 중소벤처기업부, 과기부, 서울산업진흥원 등

**출력:** `agents/outputs/정부과제_조사결과_YYYYMMDD.md`

> 결과에 적합도 상/중/하 + 신청 마감일 + 지원 금액 포함

---

### 03. 기획서 피팅 에이전트

**역할:** 특정 정부과제 공고에 맞게 기획서를 재구성. 같은 내용이지만 심사위원이 보고 싶어하는 언어로 재포장해줌

```powershell
# 대화형 (과제 직접 선택)
py agents/03_fitter.py

# 과제 지정해서 바로 실행
py agents/03_fitter.py --grant TIPS
py agents/03_fitter.py --grant 초기창업패키지
py agents/03_fitter.py --grant 창업도약패키지
py agents/03_fitter.py --grant NIPA_AI바우처

# 공고 파일 직접 붙여넣기
py agents/03_fitter.py --grant "과제명" --grant-file "공고내용.txt"
```

**내장된 과제 템플릿:**

| 키 | 과제명 | 지원 금액 |
|----|--------|-----------|
| `TIPS` | 민간투자주도형 기술창업지원 | 최대 5억 |
| `초기창업패키지` | 초기창업패키지 | 최대 1억 |
| `창업도약패키지` | 창업도약패키지 | 최대 3억 |
| `NIPA_AI바우처` | AI 바우처 지원사업 | 최대 3억 |

**출력:** `agents/outputs/피팅_{과제명}_YYYYMMDD.md`

> 공고 요구사항 분석 → 강점 매핑 → 갭 분석 → 최적화된 제안서 → 제출 체크리스트

---

### 04. 심사위원 페르소나 에이전트

**역할:** 제출 전 사전 검토. 실제 심사위원 3명이 채점하듯이 약점을 찾아줌

```powershell
# 피팅된 문서 검토 (권장)
py agents/04_evaluator.py --target "agents/outputs/피팅_TIPS_20260317.md" --grant TIPS

# 원본 기획서 검토
py agents/04_evaluator.py
```

**심사위원 페르소나:**

| 페르소나 | 프로필 | 주요 평가 항목 |
|----------|--------|----------------|
| 기술 심사위원 | ETRI 출신 AI 전문가 | 기술 혁신성, 실현 가능성, 차별성 |
| 사업화 심사위원 | 투자심사역 출신 | 시장 규모, 수익성, BEP, CAC |
| 정책 심사위원 | 중기부 출신 | 고용 창출, 사회적 가치, 정책 부합성 |

**출력:** `agents/outputs/심사평_{과제명}_YYYYMMDD.md`

> 항목별 점수 + 탈락 위험 요소 + 보완 우선순위 + 예상 심사 질문 5개 + 합격 가능성 %

---

### 05. 예산 설계 에이전트

**역할:** 과제별 예산 규정(인건비 비율, 현물부담, 간접비 한도 등)을 자동 적용해서 예산 계획서 초안 생성. 규정 위반으로 탈락하는 일을 막아줌

```powershell
# 대화형
py agents/05_budget.py

# 과제명 + 사업비 지정
py agents/05_budget.py --grant TIPS --total 200000000
py agents/05_budget.py --grant 초기창업패키지 --total 100000000
py agents/05_budget.py --grant 창업도약패키지 --total 300000000
```

**내장된 예산 규정:**

| 과제 | 인건비 한도 | 간접비 한도 | 현물부담 |
|------|------------|------------|---------|
| TIPS | 50% 이내 | 직접비의 5% | 20% 이상 |
| 초기창업패키지 | 40% 이내 | 정부지원금의 5% | 20% 이상 |
| 창업도약패키지 | 50% 이내 | 직접비의 5% | 10% 이상 |

**출력:** `agents/outputs/예산계획_{과제명}_YYYYMMDD.md`

> 비목별 배분 → 월별 집행 계획 → 인건비/외주비 명세 → 규정 위반 체크리스트

## 권장 워크플로우

```
1단계  py agents/orchestrator.py
       → 기획서 구체화부터 예산까지 순서대로 진행

2단계  agents/outputs/ 폴더에서 생성된 파일 확인
       → 심사평 기반으로 약점 파악

3단계  py agents/01_planner.py
       → 약점 보완 내용 기획서에 추가

4단계  py agents/03_fitter.py --grant [최종 선택 과제]
       → 보완된 기획서로 재피팅

5단계  py agents/05_budget.py --grant [과제명] --total [금액]
       → 예산 계획서 최종 완성
```

---

## 출력 파일 정리

모든 결과물은 `agents/outputs/` 폴더에 저장됨

| 파일명 패턴 | 생성 에이전트 | 내용 |
|------------|--------------|------|
| `기획서_vYYYYMMDD_HHMM.md` | 01_planner | 구체화된 기획서 버전 |
| `정부과제_조사결과_YYYYMMDD.md` | 02_researcher | 신청 가능 과제 전체 목록 |
| `피팅_{과제명}_YYYYMMDD.md` | 03_fitter | 공고별 최적화 제안서 |
| `심사평_{과제명}_YYYYMMDD.md` | 04_evaluator | 3인 심사위원 사전 검토 결과 |
| `예산계획_{과제명}_YYYYMMDD.md` | 05_budget | 예산 계획서 초안 |
