- **시스템 앱 블랙리스트 확장** (tracker.js + database.js)
  - 24개 Windows 시스템 프로세스 자동 필터 (explorer, svchost, dwm, cortana 등)
- **경력 기록 삭제 + 재생성** (CareerResult.jsx)

  - 삭제: 인라인 확인 후 DB에서 제거
- **GitHub Releases 자동 업데이트 인프라** (package.json, forge.config.js, main.js)
  - `update-electron-app` 패키지 추가
  - 레포 공개 전환 시 주석 해제만 하면 즉시 활성화되도록 준비
  - 현재는 베타 기간 비공개 레포로 비활성 상태
