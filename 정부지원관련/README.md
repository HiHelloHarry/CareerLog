# 정부지원관련

CareerLog 정부 창업지원사업 신청 자료 모음.
Claude Code 에이전트 파이프라인으로 생성. API 비용 없음.

---

## 폴더 구조

```
outputs/
  공통_{작성일}/                       ← 정부과제 조사결과, 사업계획서, 처리요약
  {과제명}_{접수시작}-{마감}/           ← 접수 기간이 확정된 과제
  {과제명}_{분기/시기 임시표기}/        ← 아직 공고 미확정인 과제
```

### 현재 폴더 목록

| 폴더 | 접수 기간 | 상태 |
|------|---------|------|
| `공통_20260324` | — | 조사결과·사업계획서·처리요약 |
| `예비창업패키지_20260306-20260326` | 2026.03.06 ~ 03.26 | 신청 가능 (마감 임박) |
| `TIPS_2026Q2예정` | 2026년 2분기 예정 | 공고 미확정 (임시표기) |
| `초기창업패키지_2027Q1대비` | 2027년 1월 예정 | 2027년 대비 준비본 |
| `Pre-TIPS_2026수시` | 수시 (차수별) | 공고 미확정 (임시표기) |

> 공고 날짜 확정 시 폴더명 업데이트 필요.

## 각 폴더 내 파일 구성

| 파일 | 내용 |
|------|------|
| `피팅_{과제명}_YYYYMMDD.md` | 해당 과제 평가 기준으로 re-framing한 제안서 |
| `심사평_{과제명}_YYYYMMDD.md` | 가상 심사위원 3인 평가 (점수 + 합/불합) |
| `최종_{과제명}_YYYYMMDD.md` | 피드백 반영 최종 확정본 (제출용) |

---

## 에이전트 파이프라인

Claude Code 채팅창에 말하면 자동 실행.

```
1단계  "정부과제 조사해줘"         → 공통 폴더에 조사결과 저장
2단계  "사업계획서 만들어줘"        → 기획서.md 기반 변환
3단계  "○○패키지 피팅해줘"         → 과제별 폴더에 피팅본 저장
4단계  "심사해줘"                  → 심사평 저장
5단계  "피드백 반영해서 확정해줘"   → 최종본 저장
```

> 주의: 1단계 조사 시 반드시 열린 키워드로 전체 공고 발굴 후 기존 목록 확인.
> 조사 결과가 전부라고 단정하지 말 것 — 신규 사업 누락 가능성 항상 존재.
