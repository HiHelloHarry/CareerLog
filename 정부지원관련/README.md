# CareerLog 정부과제 에이전트 파이프라인

Claude Code Agent 기반. API 비용 없음.

## 에이전트 목록

| 에이전트 | 역할 | 호출 키워드 |
|---------|------|-----------|
| `grant-researcher` | 현재 신청 가능한 정부지원과제 전수 조사 | "정부과제 조사해줘" |
| `biz-planner` | 기획서.md → 사업계획서 변환 | "사업계획서 만들어줘" |
| `grant-fitter` | 과제별 맞춤 피팅 | "TIPS 피팅해줘" |
| `grant-reviewer` | 가상 심사위원 3인 평가 | "심사해줘" |
| `grant-reviser` | 합/불합 기반 수정·확정 | "피드백 반영해줘" |

## 권장 실행 순서

```
1단계: "정부과제 조사해줘"
2단계: "기획서를 사업계획서로 만들어줘"
3단계: "TIPS, 초기창업패키지, NIPA AI바우처 피팅해줘"
4단계: "피팅된 것들 심사해줘"
5단계: "심사 결과 반영해서 수정·확정해줘"
```

## 출력 파일 위치

모든 결과물 → `agents/outputs/`

| 파일 패턴 | 내용 |
|-----------|------|
| `정부과제_조사결과_YYYYMMDD.md` | 신청 가능 과제 목록 |
| `사업계획서_YYYYMMDD.md` | 변환된 사업계획서 |
| `피팅_{과제명}_YYYYMMDD.md` | 과제별 피팅 제안서 |
| `심사평_{과제명}_YYYYMMDD.md` | 심사위원 평가 결과 |
| `최종_{과제명}_YYYYMMDD.md` | 합격 확정본 |
| `수정본_{과제명}_YYYYMMDD.md` | 불합격 후 수정본 |
| `처리요약_YYYYMMDD.md` | 전체 결과 요약 |
