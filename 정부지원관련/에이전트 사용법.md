# CareerLog 정부과제 에이전트 사용법

> 최종 업데이트: 2026-03-24
> Python 스크립트 방식 → **Claude Code 에이전트 방식으로 전면 교체**
> API 비용 없음. Claude Code 세션 안에서 직접 실행.

---

## 개요

이 에이전트 시스템은 CareerLog의 정부 창업지원사업 신청을 돕는 AI 파이프라인이다.
Claude Code의 서브에이전트로 동작하며, **"파이프라인 돌려줘"라고 말하면 알아서 처음부터 끝까지 실행**한다.

---

## 에이전트 목록

에이전트 정의 파일 위치: `C:\Users\diverse\.claude\agents\`

| 에이전트 ID | 역할 | 호출 키워드 예시 |
|------------|------|----------------|
| `grant-researcher` | 현재 신청 가능한 정부지원과제 웹 조사 | "정부과제 조사해줘" |
| `biz-planner` | 기획서.md → 정부과제용 사업계획서 변환 | "사업계획서 만들어줘" |
| `grant-fitter` | 특정 과제 공고에 맞게 사업계획서 피팅 | "예창패 피팅해줘" |
| `grant-reviewer` | 가상 심사위원 3인 모의 심사 + 합/불합 판정 | "심사해줘" |
| `grant-reviser` | 심사 결과 기반 수정·확정 + 처리 요약 | "피드백 반영해줘" |

---

## 파이프라인 구조

```
기획서.md
    ↓
[1단계] grant-researcher
        웹 검색 → 현재 신청 가능 과제 전수 조사
        출력: 정부과제_조사결과_YYYYMMDD.md
    ↓
[2단계] biz-planner
        기획서.md 읽기 → 정부과제용 사업계획서 작성
        출력: 사업계획서_YYYYMMDD.md
    ↓
[3단계] grant-fitter
        과제별 평가 기준 언어로 re-framing
        출력: 피팅_{과제명}_YYYYMMDD.md (과제 수만큼 생성)
    ↓
[4단계] grant-reviewer
        심사위원 A(VC) · B(CTO) · C(창업멘토) 3인 독립 평가
        100점 만점, 평균 70점 이상 합격
        출력: 심사평_{과제명}_YYYYMMDD.md
    ↓
[5단계] grant-reviser
        합격 → 피드백 반영 후 최종본 확정
        불합격 → 필수 보완 3항목 수정 후 재심 권고
        출력: 최종_{과제명}_YYYYMMDD.md / 처리요약_YYYYMMDD.md
```

## 사용 방법

### 전체 파이프라인 한 번에

Claude Code 채팅창에 그냥 말하면 됨:

```
1단계부터 마지막 단계까지 수행해
```

```
예창패 파이프라인 돌려줘
```

```
TIPS, 초기창업패키지 피팅하고 심사까지 해줘
```

### 단계별 개별 실행

```
정부과제 조사해줘                        → 1단계만
기획서를 사업계획서로 만들어줘            → 2단계만
TIPS에 맞게 피팅해줘                     → 3단계 (TIPS만)
피팅된 것들 심사해줘                     → 4단계
심사 결과 반영해서 최종 확정해줘          → 5단계
```

### 특정 과제만 추가로 돌리기

```
예창패도 피팅하고 심사까지 해줘
```

## 심사위원 페르소나

4단계에서 아래 3인이 독립적으로 심사한다.

| 심사위원 | 배경 | 주요 관심사 | 특징 |
|---------|------|-----------|------|
| **A — 투자심사역** | 국내 중견 VC 10년 | 시장 크기, 수익 모델, Exit | 냉정·직설적. 수치 없으면 감점 |
| **B — 기술 전문가** | SW 스타트업 CTO 출신, 겸직교수 | 기술 차별성, 구현 가능성, IP | 분석적. 세부 질문 많음 |
| **C — 창업 멘토** | 연쇄창업가 (성공·실패 경험) | 창업자 진정성, 고객 검증, 실행력 | 따뜻하지만 현실적 |

**합격 기준:** 3인 평균 70점 이상 (100점 만점)

## 주의사항

- 에이전트는 `기획서.md` 내용을 기반으로 작성한다 — **기획서.md 최신 유지 필수**
- `[창업자 확인 필요: XXX]` 표시 항목은 에이전트가 만들 수 없는 정보 (인터뷰 결과, 실측 수치 등) — 직접 채울 것
- 과제별 실제 공고 요건은 K-Startup(k-startup.go.kr)에서 최종 확인 필수
- 에이전트 정의 파일 위치: `C:\Users\diverse\.claude\agents\` (grant-researcher / biz-planner / grant-fitter / grant-reviewer / grant-reviser)
