## 💡 기타 증빙자료 — 제출 권장 항목

사업계획서의 신뢰도를 높이기 위해 아래 자료를 PDF로 묶어 제출을 권장합니다.

### 1. 제품 개발 증빙
- [ ] **GitHub 리포지토리 스크린샷** — 커밋 히스토리, 코드 규모 (CareerLog 리포)
- [ ] **앱 실행 화면 캡처** — Home, Timeline, CareerResult, Dashboard 각 1장
- [ ] **v0.7 기록 정확도 기능 시연 캡처** — 앱별 파싱, URL 추출, 활동 감지, Flow 분석
- [ ] **경력 기록 리디자인 프리뷰 캡처** — career-result-preview.html 화면

### 2. 시장 검증 증빙
- [ ] **설문조사 원본 데이터** — Google Forms 응답 시트 (n=34, 9개 직군)
- [ ] **설문 결과 요약 1페이지** — 핵심 수치 4개: 75% 미관리, 59% 수치화 어려움, 71% 사용의향, 74% 유료의향

### 3. 기술 역량 증빙 (선택)
- [ ] **개인 포트폴리오/이력서** — 풀스택 개발 경력 증빙
- [ ] **특허/지식재산권 출원 서류** — 해당 시

## 📌 제출 시 주의사항

1. **개인정보 마스킹 필수**
   - 성명: ○○○ (또는 변○○)
   - 대학교명, 직장명: 삭제
   - 학력(학과명), 직업, 수행 업무만 기재 가능

2. **파일 형식**
   - 사업계획서: 공식 양식(.docx) 그대로 제출
   - 증빙서류: PDF 권장 (여러 장이면 하나의 PDF로 합치기)
   - 파일명에 한글·특수문자 포함 가능하나 간결하게

3. **용량 제한**
   - K-Startup 시스템 파일 업로드 용량 확인 후 첨부
   - 이미지는 적절히 압축 (각 1MB 이하 권장)
