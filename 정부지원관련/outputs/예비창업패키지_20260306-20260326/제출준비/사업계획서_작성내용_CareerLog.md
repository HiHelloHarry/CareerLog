## □ 창업 아이템 개요(요약)

### 명칭
> CareerLog

### 범주
> AI 기반 경력 자동 기록 데스크톱 서비스

### 아이템 개요
> 직장인이 업무 중 사용하는 PC의 포그라운드 앱·문서·URL을 자동 감지하여 활동 타임라인을 생성하고, AI가 이를 이력서용 STAR(Situation-Task-Action-Result) 포맷의 경력 기록으로 자동 변환해주는 Windows 데스크톱 앱. 사용자는 업무 시작 버튼을 누르기만 하면 되며, 이직 시 경력기술서·이력서에 바로 복사·붙여넣기할 수 있는 문장을 자동 생성한다. 모든 데이터는 사용자 PC에만 저장(로컬 우선 설계)되어 보안 우려를 해소한다.
>
> 핵심 기능: (1) Win32 API 기반 PC 활동 자동 감지 (10초 폴링, 앱별 파서 17개), (2) AI STAR 경력 기록 자동 생성 (직군·직급 개인화), (3) 커리어 대시보드 (히트맵, 스트릭, 인사이트), (4) Markdown/HTML 내보내기·클립보드 복사
>
> 고객 제공 혜택: 경력 정리에 소요되는 시간을 "며칠"에서 "버튼 한 번"으로 단축. 자체 설문(n=34)에서 응답자 71%가 긍정적 사용 의향, 74%가 유료 지불 의사 확인.

### 문제 인식 (Problem)
> 직장인은 일을 하면서도 경력을 쌓지 못한다. 이직 준비 시 "어떤 프로젝트를 했지?"라는 질문 앞에서 75%가 상시 경력 관리를 하지 않고(자체 설문 n=34), 55%가 경력 정리에 하루 이상을 소비한다. 성과를 수치로 표현하기 어려운 것이 1위 페인포인트(59%)로, 기존 노션·엑셀·AI 이력서 툴은 모두 "이미 기억하는 것"을 입력해야 하며, 기억 자체를 자동으로 쌓아주는 서비스는 국내외에 없다.

### 실현 가능성 (Solution)
> MVP v0.7 개발 완료 상태. Windows x64 Setup.exe 인스톨러 배포 가능. GitHub 공개 저장소에서 전체 소스 코드·커밋 이력 확인 가능(https://github.com/HiHelloHarry/CareerLog). 협약기간 내 (1) 베타 유저 100명 확보 및 리텐션 검증, (2) B2C 결제 시스템 구축, (3) Mac 버전 개발을 최종 산출물로 완성할 계획.

### 성장전략 (Scale-up)
> 베타 유저 리텐션 검증 → Pro 구독 런칭(월 4,900원) → 채용 플랫폼 파트너십 → B2B 확장. 자체 설문에서 유료 의향 74%, 월 5,000원 이하 수용률 65% 확인. BEP 유료 구독자 약 300명 시점. 국내 HR테크 시장 CAGR 7.55%(IMARC 2024), "경력 자동 축적" 카테고리 선점 기회.

### 팀 구성 (Team)
> 대표자 1인이 기획·개발·QA·배포 전 과정 단독 완수. 기획/QA/PM 멀티롤 실무 경력 보유. 협약기간 내 기술 고문(비상근) 확보 예정, 공동창업자(그로스) 탐색 중(후보 2명 대화 진행 중).

### 이미지
> (GitHub Releases 스크린샷, 앱 타임라인 화면, AI STAR 생성 결과물 캡처 삽입)

## 1. 문제 인식 (Problem) — 창업 아이템의 필요성

### 국내·외 시장 현황 및 문제점

**1) 직장인 경력 관리의 구조적 실패**

직장인은 일을 하면서도 경력을 쌓지 못한다. 이것은 의지가 아니라 구조의 문제다.

- 국내 경영·회계 관련 사무직 임금근로자 **426만 2천 명** (통계청 2024년 상반기 지역별고용조사, 직업중분류 전체 1위)
- 20~40대 정규직 근로자 **69.5%가 이직 고려** 중 (한국경영자총협회 「근로자 이직 트렌드 조사」, 2024.04, 글로벌리서치 위탁, n=1,500)
- 국내 취업자 업무용 생성형 AI 활용률 **51.8%** (한국은행 BOK 이슈노트 제2025-22호, 2025.08), 미국(26.5%) 대비 약 2배

**2) 잠재 고객 설문으로 확인된 문제의 실체 (n=34, 2026.03.25, 자체 Google Forms 설문)**

| 핵심 지표 | 수치 | 시사점 |
|---|---|---|
| 이직할 때만 or 분기 이하로 경력 정리 | **75%** | "자동 기록" 가치 명확 |
| 이력서 작성 시 "수치 표현 어렵다" | **59%** (1위) | 수치 자동 추출 기능 수요 |
| 경력 정리에 하루 이상 소요 | **55%** | 시간 절약 가치 검증 |
| "바로 써보고 싶다" + "관심 있다" | **71%** | 서비스 수용도 높음 |
| 유료 지불 의사 | **74%** | WTP 검증 |
| 핵심 연차 3~5년차 | **36%** (최다) | 타겟 프로파일 일치 |
| 이직 관련 활성 상태 | **65%** | 경총 69.5%와 교차 검증 |

> 설문 대상: 기획/PM(26%), 디자인(24%), 개발(18%), QA(9%) 등 9개 직군, 경력 1년 미만~10년 이상 분포. Google Forms 원본 데이터(CSV) 보유.

**3) 기존 솔루션의 공통 맹점**

| 기존 솔루션 | 방식 | 근본 한계 |
|---|---|---|
| 노션 / 메모 앱 | 직접 입력 | 기록하려면 업무를 멈춰야 함 |
| 링크드인 | 직접 작성 | 과거 회고 방식, 기억 의존 |
| Toggl 등 타임트래킹 | 수동 타이머 | 시작/종료 직접 입력, 성과 해석 불가 |
| Resume.io, Rezi 등 | 입력 기반 생성 | 입력할 내용 자체가 없으면 무용 |

설문에서도 현재 경력 기록 방법은 "노션/메모(38%)", "엑셀(32%)", "기억에 의존(18%)" 순으로, 자동화 대안 없이 수동 방식에 머물러 있음이 확인됐다.

**핵심 공백:** "경력을 자동으로 축적"하는 카테고리 자체가 국내외 시장에 없다.

### 문제 해결을 위한 창업 아이템의 개발 필요성

기존 도구는 "이미 기억하는 것"을 잘 쓰도록 도와줄 뿐이다. CareerLog는 기억 자체를 자동으로 쌓아주는 유일한 서비스로, 업무 중 PC 활동을 자동 감지하여 이직 시 바로 활용 가능한 경력 기록을 생성한다. 설문 결과 "업무 활동 자동 기록"이 가장 끌리는 기능 1위(47%)를 차지했으며, 이 문제의 해결이 돈을 낼 만한 가치인지에 대해 74%가 유료 지불 의사를 보였다.

### 창업 아이템의 차별성 및 경쟁력 확보 전략

**1. 입력 없는 자동 기록** — 국내 유일
시중 모든 도구는 사용자의 능동적 입력을 전제로 한다. CareerLog는 PC 활동을 수동 개입 없이 자동 감지한다. 설문에서 "업무 활동 자동 기록"이 가장 끌리는 기능 1위(47%).

**2. 로컬 우선 프라이버시 설계**
활동 데이터는 전량 사용자 PC에만 저장. 설문 자유 의견에서 보안 우려가 15%로 반복 등장했으며, CareerLog의 로컬 저장 + 앱 블랙리스트 + URL depth 제한 + 민감 도메인 차단 기능이 이 우려에 직접 대응.

**3. 기록 정확도 기술적 해자(Moat)**
Win32 API 직접 호출(네이티브 모듈 의존 없음), 앱별 파서 17개, 브라우저 URL 추출, 키보드·마우스 활동 감지, 작업 흐름(Flow) 감지. UIAutomation 방식의 창 제목 오염 버그를 직접 발견·수정한 독자적 구현.

**4. 글로벌 경쟁 포지셔닝**

| 서비스 | 자동 기록 | AI 경력 변환 | 로컬 저장 | 한국어 | 이직 특화 |
