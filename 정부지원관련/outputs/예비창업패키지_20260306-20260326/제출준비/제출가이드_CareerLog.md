## STEP 1. 사업계획서 작성 (docx 양식 채우기)

### 준비물
- `[별첨 1] 2026년 예비창업패키지 사업계획서 양식.docx` — 공식 양식
- `사업계획서_작성내용_CareerLog.md` — 작성 완료된 내용

### 작성 방법
1. **공식 양식 열기** → 기존 안내 문구(파란 글씨) 전부 삭제
2. `사업계획서_작성내용_CareerLog.md`의 각 섹션을 해당 칸에 **복사·붙여넣기**
3. 표(table)는 Word에서 직접 정리 (마크다운 표 → Word 표로 변환)
4. **글씨체/크기**: 양식에 지정된 서식 유지 (보통 맑은고딕 10~11pt)
5. **분량**: 각 항목별 제한 분량 준수 (양식에 표기된 페이지 수)

### 섹션별 매핑

| 양식 섹션 | 내용 파일 위치 |
|-----------|---------------|
| □ 일반현황 | `## □ 일반현황` |
| □ 개요(요약) | `## □ 개요(요약)` |
| 1. 문제인식 (Problem) | `## 1. 문제인식 (Problem)` |
| 2. 실현가능성 (Solution) | `## 2. 실현가능성 (Solution)` |
| 3. 성장전략 (Scale-up) | `## 3. 성장전략 (Scale-up)` |
| 4. 팀 구성 (Team) | `## 4. 팀 구성 (Team)` |

### 주의사항
- **개인정보 마스킹**: 성명→○○, 학교명·직장명 삭제 (학과명·직업·업무는 OK)
- **이미지 삽입**: 앱 스크린샷, 설문 차트 등은 Word에 직접 삽입
- **사업비 금액**: 1단계 2,000만원 + 2단계 2,000만원 = 총 4,000만원으로 통일
