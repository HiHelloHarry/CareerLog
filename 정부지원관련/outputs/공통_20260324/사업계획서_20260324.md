## 1. 문제 정의 (Problem)

### 해결하려는 문제: 직장인은 자신의 경력을 증명할 기록이 없다

대부분의 직장인은 매일 업무를 수행하지만, 이직·연봉 협상·성과 평가 시점에 자신이 한 일을 구체적으로 설명할 증거를 갖고 있지 않다. "3년 동안 뭘 했는지" 떠올려야 하는 순간, 기억에만 의존해야 한다.

- 이직 결심 후 경력 정리에 상당한 시간이 소요되며, 대부분 직함과 회사명만 쓴 빈 이력서로 마무리된다.
- 연봉 협상·성과 평가 자리에서 수치로 된 근거가 없어 불리하게 협상한다.
- 결국 문제는 이력서 작성 능력이 아니라, **기록되지 않은 경험**이다.

### 기록이 없는 구조적 이유

기록하려면 행동을 멈춰야 한다. 업무 중 별도로 기록하는 행위 자체가 부담이기 때문에, 대부분의 직장인은 기록을 시작조차 하지 않는다. 노션·옵시디언 같은 도구가 있어도 결국 안 쓰게 되는 이유다.

### 기존 솔루션의 한계

| 도구 | 한계 |
|------|------|
| 노션·옵시디언 | 기록은 가능하나 구조화·자동화 없음. 지속 사용률 낮음 |
| 링크드인 | 직접 수동 입력 필요. 업데이트 방치 패턴이 고착 |
| AI 이력서 툴 (Resume.io, Rezi 등) | 작성 도움만 제공. 기억 자체가 없으면 소용없음 |
| 타임트래킹 툴 (Toggl 등) | 시간만 기록. "어떤 성과였는지" 해석 불가 |
| 사람인·잡코리아 | 채용 플랫폼. 이직 결심 이후에야 사용하는 구조 |

**핵심 공백:** 기존 시장에 "경력을 자동으로 축적"하는 카테고리 자체가 없다.

### 타겟 고객의 페인포인트

메인 타겟: **직장인 3~7년차, 이직 예비자**

- 경력은 있으나 정리가 안 된 기획자·PM·디자이너·개발자
- 지금 당장 이직할 생각은 없지만 "나중에 필요하겠다"는 것을 아는 사람
- 연봉 협상·성과 평가에서 수치 근거 없이 밀린 경험이 있는 사람

타겟의 핵심 심리: "지금 쓰는 것이 아니라 **쌓아두는 것**"이다. CareerLog는 이직 도구가 아니라 경력 보험이다.

## 2. 솔루션 (Solution)

### CareerLog — PC가 알아서 기록하는 업무 블랙박스

CareerLog는 Windows 데스크톱 앱(Electron)이다. 사용자가 별도로 입력하지 않아도 된다.

"업무 시작" 버튼 하나로 PC가 블랙박스로 변한다. 어떤 앱을 얼마나 썼는지, 어떤 프로젝트에 시간을 쏟았는지 10초마다 자동으로 기록된다. 업무가 끝나면 AI가 그 기록을 STAR 포맷 경력 기술서로 변환한다.

```
업무 시작 버튼 클릭
    ↓
PC 포그라운드 창 자동 감지 (10초 간격)
    ↓
타임라인 자동 생성 (앱별·시간순, 유휴 시간 제외)
    ↓
업무 종료 → 프로젝트 태깅 (선택)
    ↓
AI가 STAR 포맷 경력 기록 자동 생성
    ↓
경력 대시보드 누적 → 이직·협상·평가 시 꺼내 씀
```

사용자가 하는 일은 단 두 가지다 — 시작 버튼, 그리고 이직할 때 복사 붙여넣기.

### 핵심 기능

**자동 기록**
- PC 포그라운드 창을 10초마다 감지 (Win32 GetForegroundWindow API 기반)
- 15분 이상 자리 비움 시 Idle로 자동 처리하여 기록에서 제외
- 세션 단위 기록 및 프로젝트 태깅 (8개 작업 유형 칩 선택 가능)
- AI가 활동 내용에서 성과 수치를 자동 하이라이트

**타임라인 조회**
- 앱별 그룹 뷰 / 시간순 토글
- 날짜 피커로 과거 세션 조회
- 활동 기록 수동 삭제·메모 수정 가능

**AI 경력 기록 생성**
- 세션 활동 기반으로 Situation·Task·Action·Result(STAR) 포맷 자동 작성
- 직군·직급·기술스택 설정을 AI 프롬프트에 자동 반영 → 직무 맞춤 서술
- Markdown·HTML 내보내기, 클립보드 복사 (경력기술서·STAR·LinkedIn 형식)

**커리어 대시보드**
- 이번 주 활동 히어로 카드, 누적 카드 4개 (업무 일수·경력기록 수·총 시간·사용 앱)
- 최근 12주 히트맵 (셀 클릭 시 해당 날짜 타임라인으로 이동)
- 집중 스트릭 카운터 (연속 기록 일수, 7일 이상 골드 배지)
- 오늘의 인사이트 카드 (매일 새로운 로컬 분석 인사이트)

**프라이버시**
- 모든 데이터는 사용자 PC에만 저장. 외부 서버 전송 없음.
- 온보딩 화면에 "모든 기록은 이 PC에만 저장" 명시

### 차별화 포인트

경쟁사들은 모두 "이미 기억하고 있는 것"을 잘 쓰게 도와준다. CareerLog는 "기억할 필요 없게" 만든다.

```
              [기록 자동화 높음]
                    │
                    │   ★ CareerLog
                    │     (AI 자동 축적 + 출력)
                    │
 [이력서 출력 약함] ──────┼────── [이력서 출력 강함]
                    │
        노션·옵시디언│         Resume.io · Rezi
        (수동 기록)  │         (1회성 작성 도움)
                    │
              [기록 자동화 낮음]
```

CareerLog는 "기록 자동화"와 "출력 품질"을 동시에 해결하는 포지션이며, 이 포지션에 해당하는 경쟁 서비스가 현재 국내에 없다.

### 글로벌 경쟁자 비교

해외에는 화면·활동 기반 자동 기록 서비스가 존재한다. CareerLog는 이들과 카테고리가 겹치지 않는다.

| 서비스 | 자동 기록 | AI 경력 변환 | 로컬 저장 | 한국어 지원 | 국내 이직 최적화 | 가격 |
|--------|----------|------------|----------|-----------|---------------|------|
| **CareerLog** | ✅ | ✅ STAR 포맷 | ✅ | ✅ | ✅ | 월 9,900원 |
| MS Recall | ✅ 화면 전체 캡처 | ❌ 검색만 가능 | 온디바이스 (일부 전송 논란) | ❌ | ❌ | Copilot+ PC 전용 |
| Rewind AI | ✅ 화면 캡처 | ❌ | ✅ Mac 전용 | ❌ | ❌ | $19/월 |
| Notion·Obsidian | ❌ 수동 입력 | ❌ | 선택 가능 | ✅ | ❌ | 무료~$8 |
| Resume.io·Rezi | ❌ | ✅ 작성 보조 | ❌ | ❌ | ❌ | $6~$25/월 |

**핵심 차별점:** MS Recall은 화면을 통째로 저장하는 방식으로 개인정보 논란이 지속되고 있으며, 국내 이직 시장에 특화된 AI 경력 기록 변환 기능이 없다. Rewind AI는 Mac 전용이며 경력 문서 생성을 지원하지 않는다. CareerLog는 **한국 직장인의 이직 문서(경력기술서·STAR)에 특화된 유일한 로컬 저장 자동화 서비스**다.

### 기술 독창성

- Win32 GetForegroundWindow API 기반 PowerShell 영구 프로세스(spawn 1회, 10초 루프)로 안정적 창 감지 구현. 기존 오픈소스(active-win) 대비 Electron 패키징 환경에서의 안정성 확보.
- AI 경력 기록 생성 시 사용자 직군·직급·기술스택을 프롬프트에 주입하는 개인화 구조 설계.
- 모든 데이터를 로컬 JSON 파일로 관리하는 서버리스 아키텍처 — 프라이버시 소구 및 초기 운영비용 최소화.

### 목표 고객 세그먼트

**1순위 — 경력직 이직 예비자 (직장인 3~7년차)**
- 경력은 있으나 정리가 안 된 기획자·PM·디자이너·개발자
- "이직할 때가 되면 쓰겠다"가 아니라 "지금부터 쌓아두어야겠다"는 인식을 가진 층

**2순위 — 신입 취준생**
- 인턴·프로젝트 경험을 체계적으로 기록하고 싶은 층

**3순위 — 프리랜서·N잡러**
- 다수 클라이언트 프로젝트를 관리하고 포트폴리오를 축적해야 하는 층
