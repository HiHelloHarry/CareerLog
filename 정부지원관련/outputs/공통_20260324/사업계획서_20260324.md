## 1. 문제 정의 (Problem)

### 해결하려는 문제: 직장인은 귀찮아서 경력 관리를 안 한다

대부분의 직장인은 매일 업무를 수행하지만, 경력 관리를 하지 않는다. 이직·연봉 협상·성과 평가 시점에 "3년 동안 뭘 했는지" 떠올려야 하는 순간, 기억에만 의존해야 한다. 실제로 직장인 대상 설문(n=34)에서 88%가 경력 관리를 전혀 하지 않는다고 응답했다.

- 이직 결심 후 경력 정리에 상당한 시간이 소요되며, 대부분 직함과 회사명만 쓴 빈 이력서로 마무리된다.
- 연봉 협상·성과 평가 자리에서 수치로 된 근거가 없어 불리하게 협상한다.
- 결국 문제는 이력서 작성 능력이 아니라, **귀찮아서 쌓이지 않은 경력 기록**이다.

### 기록이 귀찮은 구조적 이유

기록하려면 행동을 멈춰야 한다. 업무 중 별도로 기록하는 행위 자체가 부담이기 때문에, 대부분의 직장인은 기록을 시작조차 하지 않는다. 노션·옵시디언 같은 도구가 있어도 결국 안 쓰게 되는 이유다.

### 기존 솔루션의 한계

| 도구 | 한계 |
|------|------|
| 노션·옵시디언 | 기록은 가능하나 구조화·자동화 없음. 지속 사용률 낮음 |
| 링크드인 | 직접 수동 입력 필요. 업데이트 방치 패턴이 고착 |
| AI 이력서 툴 (Resume.io, Rezi 등) | 작성 도움만 제공. 기억 자체가 없으면 소용없음 |
| 타임트래킹 툴 (Toggl 등) | 시간만 기록. "어떤 성과였는지" 해석 불가 |
| 사람인·잡코리아 | 채용 플랫폼. 이직 결심 이후에야 사용하는 구조 |

**핵심 공백:** 기존 시장에 "귀찮음 제로로 매일 경력이 쌓이는" 카테고리 자체가 없다.

### 타겟 고객의 페인포인트

메인 타겟: **직장인 3~7년차, 이직 예비자**

- 경력은 있으나 정리가 안 된 기획자·PM·디자이너·개발자
- 지금 당장 이직할 생각은 없지만 "나중에 필요하겠다"는 것을 아는 사람
- 연봉 협상·성과 평가에서 수치 근거 없이 밀린 경험이 있는 사람

타겟의 핵심 심리: "경력 관리가 중요한 건 아는데, 귀찮아서 안 한다." 설문(n=34)에서 88%가 경력 관리를 하지 않는다고 답했다. CareerLog는 이 귀찮음을 제거하여 버튼 두 개만으로 매일 커리어가 쌓이게 만드는 도구다.

## 2. 솔루션 (Solution)

### CareerLog — 버튼 두 개로 매일 커리어가 자동으로 쌓이는 도구

CareerLog는 Windows 데스크톱 앱(Electron)이다. 사용자가 별도로 입력하지 않아도 된다.

"업무 시작"과 "업무 종료" — 이 두 버튼만 누르면 매일의 업무가 자동으로 기록되고, 커리어가 쌓인다. 어떤 앱을 얼마나 썼는지, 어떤 프로젝트에 시간을 쏟았는지 10초마다 자동으로 감지된다. 쌓인 기록은 커리어 대시보드에서 확인하고, 필요할 때 AI가 STAR 포맷으로 정리해준다.

```
업무 시작 버튼 클릭
    ↓
PC 포그라운드 창 자동 감지 (10초 간격, 앱별 파서 17개)
    ↓
작업 흐름(Flow) 자동 분석 — 개별 앱 전환을 의미 단위로 묶음
    ↓
타임라인 자동 생성 + 매일 커리어 대시보드에 누적
    ↓
업무 종료 버튼 클릭 → 프로젝트 태깅 (선택)
    ↓
필요할 때 AI가 STAR 포맷 경력 기록 생성
    ↓
이직·협상·평가 시 꺼내 씀
```

업무 시작, 업무 종료 — 두 버튼만으로 매일 커리어가 쌓인다.

### 핵심 기능

**1. 자동 기록 — 매일 커리어가 쌓이는 핵심**
- PC 포그라운드 창을 10초마다 감지 (Win32 GetForegroundWindow API 기반)
- 앱별 창 제목 파서 17개 (VS Code, Chrome, Edge, Slack, Figma, Notion, Excel, Word, PPT, Zoom, Teams, KakaoTalk, Postman, IntelliJ, WebStorm, Terminal 등) — 구조화된 메타데이터 자동 추출
- 브라우저 URL 추출 (UI Automation) — 민감 도메인은 호스트만, 일반 도메인은 pathname depth 2까지만 저장
- 키보드+마우스 활동 감지 → inputLevel (active/light/passive) 자동 판정
- 작업 흐름(Flow) 감지 — 개별 앱 전환을 의미 단위로 묶어 맥락 보존
- 15분 이상 자리 비움 시 Idle로 자동 처리하여 기록에서 제외
- 세션 단위 기록 및 프로젝트 태깅 (8개 작업 유형 칩 선택 가능)

**2. 커리어 대시보드 — 쌓인 기록을 한눈에**
- 3단 레이아웃: 프로필+대시보드 통계 / 월별 캘린더 / 업무 카드 그리드
- 프로필 카드 + 이번 주·총 업무일·경력기록 수 통계 + 주간 바차트 + TOP 도구
- 월별 캘린더에 기록 있는 날짜에 프로젝트 태그 표시, 클릭 시 카드 필터링
- 최근 12주 히트맵 (셀 클릭 시 해당 날짜 타임라인으로 이동)
- 집중 스트릭 카운터 (연속 기록 일수, 7일 이상 골드 배지)
- 오늘의 인사이트 카드 (매일 새로운 로컬 분석 인사이트)

**3. AI STAR 경력 기록 — 필요할 때 꺼내 쓰는 도구**
- 세션 활동 기반으로 Situation·Task·Action·Result(STAR) 포맷 자동 작성
- Flow 기반 구조화 입력 + 핵심/보조 역할 태그로 AI 프롬프트 고도화
- 직군·직급·기술스택 설정을 AI 프롬프트에 자동 반영 → 직무 맞춤 서술
- 카드 클릭 시 오른쪽 슬라이드 드로어로 STAR/불릿 상세 확인, 키보드 좌우 이동

**4. 내보내기 — 이직·협상·평가에 바로 활용**
- Markdown·HTML 내보내기
- 클립보드 복사 3종 (경력기술서·STAR·LinkedIn 형식)
- 프로젝트 필터 + 키워드 검색으로 필요한 기록만 추출

**프라이버시**
- 모든 데이터는 사용자 PC에만 저장. 외부 서버 전송 없음.
- 온보딩 화면에 "모든 기록은 이 PC에만 저장" 명시

### 차별화 포인트

경쟁사들은 모두 "이미 기억하고 있는 것"을 잘 쓰게 도와준다. CareerLog는 "귀찮음 제로로 매일 커리어가 쌓이게" 만든다.

```
              [기록 자동화 높음]
                    │
                    │   ★ CareerLog
                    │     (자동 기록 + 커리어 관리)
                    │
 [커리어 관리 약함] ──────┼────── [커리어 관리 강함]
                    │
        Toggl·RescueTime│      Resume.io · Rezi
        (시간만 기록)    │      (1회성 작성 도움)
                    │
              [기록 자동화 낮음]
              노션·옵시디언·링크드인
              (수동 입력)
```

CareerLog는 "기록 자동화"와 "커리어 관리"를 동시에 해결하는 포지션이며, 귀찮음 제로로 매일 커리어가 쌓이는 유일한 서비스다. 이 포지션에 해당하는 경쟁 서비스가 현재 국내에 없다.

### 글로벌 경쟁자 비교

해외에는 화면·활동 기반 자동 기록 서비스가 존재한다. CareerLog는 이들과 카테고리가 겹치지 않는다.

| 서비스 | 자동 기록 | AI 경력 변환 | 로컬 저장 | 한국어 지원 | 국내 이직 최적화 | 가격 |
|--------|----------|------------|----------|-----------|---------------|------|
| **CareerLog** | ✅ | ✅ STAR 포맷 | ✅ | ✅ | ✅ | 월 4,900원 |
| MS Recall | ✅ 화면 전체 캡처 | ❌ 검색만 가능 | 온디바이스 (일부 전송 논란) | ❌ | ❌ | Copilot+ PC 전용 |
| Rewind AI | ✅ 화면 캡처 | ❌ | ✅ Mac 전용 | ❌ | ❌ | $19/월 |
| Notion·Obsidian | ❌ 수동 입력 | ❌ | 선택 가능 | ✅ | ❌ | 무료~$8 |
| Resume.io·Rezi | ❌ | ✅ 작성 보조 | ❌ | ❌ | ❌ | $6~$25/월 |

**핵심 차별점:** MS Recall은 화면을 통째로 저장하는 방식으로 개인정보 논란이 지속되고 있으며, 국내 이직 시장에 특화된 AI 경력 기록 변환 기능이 없다. Rewind AI는 Mac 전용이며 경력 문서 생성을 지원하지 않는다. CareerLog는 **귀찮음 제로로 매일 커리어가 쌓이는 유일한 로컬 저장 자동화 서비스**다.

### 기술 독창성

- Win32 GetForegroundWindow API 기반 PowerShell 영구 프로세스(spawn 1회, 10초 루프)로 안정적 창 감지 구현. 기존 오픈소스(active-win) 대비 Electron 패키징 환경에서의 안정성 확보.
- 앱별 창 제목 파서 17개 — VS Code, Chrome, Edge, Slack, Figma, Notion, Excel 등 주요 업무 앱의 창 제목에서 프로젝트명·파일명·URL 등 구조화 메타데이터를 자동 추출.
- 브라우저 URL 추출 (UIAutomationClient) — 민감정보 depth 제한으로 프라이버시 보호와 기록 정확도를 동시 확보.
- 키보드+마우스 활동 감지 (GetAsyncKeyState + GetCursorPos) → inputLevel 3단계 판정. 디자이너 등 마우스 중심 작업도 active로 정확히 분류.
- 작업 흐름(Flow) 감지 엔진 — 개별 앱 전환을 의미 단위로 묶어 AI 프롬프트에 구조화 입력. 5분 이하 이탈은 보조 태그, 이전 flow 재합류 허용.
- AI 경력 기록 생성 시 Flow 기반 핵심/보조 역할 태그 + 사용자 직군·직급·기술스택을 프롬프트에 주입하는 개인화 구조 설계.
- 모든 데이터를 로컬 JSON 파일로 관리하는 서버리스 아키텍처 — 프라이버시 소구 및 초기 운영비용 최소화.

### 사회적 가치

**직장인의 협상력 강화**
기록 없이 밀리는 연봉 협상·성과 평가 구조를 개선한다. 근로자가 자신의 기여를 수치로 입증할 수 있을 때 정당한 보상 협상이 가능해진다.

**경력 기록 격차 해소**
기록 능력이 뛰어난 사람뿐만 아니라 기록을 못하는 사람도 자동으로 경력이 쌓이는 구조를 제공한다. 기획자·디자이너·PM처럼 코드로 성과를 증명하기 어려운 직군의 경력 가시화에 기여한다.

**프라이버시 보호 기반 AI 도구 모델 제시**
모든 데이터를 로컬에만 저장하는 구조로, AI 도구 사용에 따른 개인정보 유출 우려 없이 경력 데이터를 안전하게 관리하는 모델을 제시한다.
