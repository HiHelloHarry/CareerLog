## 1. 왜 지금 이 팀이 이 문제를 풀어야 하는가

**"직장인 경력 기록 자동화"는 반드시 나올 시장이다. 우리는 이미 만들었다.**

국내 경영·회계 관련 사무직 임금근로자는 426만 2천 명이다(통계청 2024년 상반기 지역별고용조사). 이들 중 69.5%가 이직을 고려하고 있지만(한국경영자총협회 「근로자 이직 트렌드 조사」, 2024), 이직 시점에 자신이 한 일을 수치로 증명하는 직장인은 극소수다.

문제는 기억력이 아니다. **기록 구조 자체가 없다.** 업무 중 별도로 기록하는 행위는 마찰이 크기 때문에, 대부분의 직장인은 시작조차 하지 않는다. 노션·옵시디언 같은 도구가 있어도 결국 안 쓰게 되는 이유다.

기존 솔루션(노션, 링크드인, 타임트래킹)은 모두 "이미 기억하는 것"을 잘 쓰게 도와줄 뿐이다. **기억 자체를 자동으로 축적하는 서비스는 지금까지 존재하지 않았다.** CareerLog는 이 공백을 최초로 점유한다.

## 2. 제품: 무엇을 만들었는가

CareerLog는 Windows 데스크톱 앱이다. 사용자 동작은 두 가지뿐이다.

1. 업무 시작 버튼
2. 이직할 때 복사 붙여넣기

그 사이 PC 포그라운드 창을 10초마다 자동 감지하고, 업무 종료 시 AI가 STAR 포맷 경력 기록을 자동 생성한다.

**기술 핵심:** PowerShell + Win32 GetForegroundWindow() API 직접 호출로 네이티브 모듈 의존성 없이 안정적 창 감지 구현. UIAutomationClient 방식의 창 제목 오염 버그를 Win32 API 방식으로 해결하여 기록 정확도를 확보했다. Electron 패키징 환경에서 오픈소스(active-win) 대비 런타임 안정성을 직접 검증 완료.

**현재 상태 (v0.6, 2026.03.24 기준):**

| 기능 | 상태 |
|------|------|
| PC 활동 자동 감지 (10초 폴링, Idle 15분 자동 처리) | 완료 |
| 세션 시작·종료, 프로젝트 태깅 | 완료 |
| AI STAR 경력 기록 자동 생성 (직군·직급·기술스택 프롬프트 주입) | 완료 |
| 커리어 대시보드 (12주 히트맵, 스트릭, 인사이트 카드) | 완료 |
| Markdown/HTML 내보내기, 클립보드 복사 | 완료 |
| Windows x64 Setup.exe 인스톨러 (Squirrel) | 완료 |

MVP는 이미 동작한다. 투자금은 "만들기"가 아닌 "성장"에 쓰인다.

## 3. 시장 포지셔닝

```
              [기록 자동화 높음]
                    |
                    |   ★ CareerLog
                    |     (AI 자동 축적 + 출력)
                    |
 [이력서 출력 약함] ──────┼────── [이력서 출력 강함]
                    |
        노션·옵시디언|         Resume.io · Rezi
        (수동 기록)  |         (1회성 작성 도움)
                    |
              [기록 자동화 낮음]
```

링크드인, 타임트래킹 툴(Toggl 등), AI 이력서 서비스 등 기존 플레이어는 모두 "수동 기록 후 정리"를 돕는다. "기록 자동화" 축 자체가 비어 있다.

### 글로벌 경쟁자 비교

| 서비스 | 자동 기록 | AI 경력 변환 | 로컬 저장 | 한국어 지원 | 국내 이직 최적화 | 가격 |
|--------|----------|------------|----------|-----------|---------------|------|
| **CareerLog** | O | O (STAR 포맷) | O | O | O | 월 9,900원 |
| MS Recall | O (화면 전체 캡처) | X (검색만 가능) | 온디바이스 (일부 전송 논란) | X | X | Copilot+ PC 전용 |
| Rewind AI | O (화면 캡처) | X | O (Mac 전용) | X | X | $19/월 |
| Notion·Obsidian | X (수동 입력) | X | 선택 가능 | O | X | 무료~$8 |
| Resume.io·Rezi | X | O (작성 보조) | X | X | X | $6~$25/월 |

MS Recall은 화면 전체 캡처 방식으로 개인정보 논란이 지속되고 있으며, 국내 이직 문서(경력기술서·STAR)에 특화된 기능이 없다. Rewind AI는 Mac 전용이며 경력 문서 생성을 지원하지 않는다. CareerLog는 **한국 직장인의 이직 문서에 특화된 유일한 로컬 저장 자동화 서비스**다.

## 5. 비즈니스 모델 및 수익 전망

**B2C 구독 → B2B SaaS → 전략적 파트너십**

| 단계 | BM | 목표 지표 |
|------|-----|---------|
| Phase 1 (베타, 0~6개월) | 무료 배포, 리텐션 검증 | D30 리텐션 40% 이상 |
| Phase 2 (구독, 7~12개월) | Pro 월 9,900원 / Premium 월 19,900원 | MRR 500만원, MAU 1,000명 |
| Phase 3 (성장, 12~18개월) | Mac 확장, 콘텐츠 채널, JD 매칭 기능 출시 | MAU 5,000명, 유료 구독자 300명 |
| Phase 4~5 (18개월~) | 채용 플랫폼 파트너십, B2B 팀 플랜 | ARR 6억 원 이상 (추정) |

**프리미엄 구조 설계 원칙:** Free 플랜에서는 기록이 계속 쌓이되, 꺼내 쓸 때(내보내기·AI 생성)에만 유료 전환. 기록을 쌓은 사용자일수록 이탈 비용이 높아지는 Lock-in 구조.

**Exit 포지셔닝:** CareerLog가 쌓는 "업무 기록 자동화 레이어"와 이직 행동 데이터는 채용 플랫폼이 아직 보유하지 못한 자산이다. 잡코리아·원티드 등 국내 채용 플랫폼 및 글로벌 HR테크 플랫폼과의 전략적 파트너십 또는 인수가 중장기 시나리오에 포함된다.
