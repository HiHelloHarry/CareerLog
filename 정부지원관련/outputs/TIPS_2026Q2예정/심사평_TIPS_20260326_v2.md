## 핵심 피드백 요약

### 강점
1. **실제 동작하는 MVP (v0.8)**: GitHub 공개 + Setup.exe 배포까지 완료된 상태는 대부분의 TIPS 지원자보다 실행력 면에서 앞서 있음. "만들었다"가 아니라 "배포했다"는 강점
2. **명확한 문제 정의**: "기록 구조 자체가 없다"는 프레이밍이 간결하고 공감 가능. 설문 88%(분기 이하 정리)가 문제의 실재를 지지
3. **단위경제 분석의 투명성**: AI 비용 구조, 마진율, 손익분기점을 구체적 수치로 제시. 자금 사용 계획의 월별 현금흐름표도 양호

### 보완 필수 항목 (치명적 약점)

1. **[팀 역량] 1인 예비창업자 체제**: TIPS는 팀 역량에 높은 가중치를 두며, 1인 체제 + 사업자등록증 미보유는 가장 큰 감점 요인. "탐색적 대화 2명"은 실체가 없음
   - 권장 수정 방향: 공동창업자 최소 1명 확정(이름, 경력, 합류 시점, 지분 조건 명시). CTO급 기술 파트너가 아니더라도 그로스/비즈니스 파트너라도 확정. TIPS 운영사 사전 미팅 2곳 이상 완료

2. **[솔루션 차별성] 기술혁신성 논증 부재**: TIPS 핵심 심사 기준인 "기술혁신성"에 대한 답이 없음. Win32 API + 규칙 기반 파서 + LLM API 호출의 조합은 기존 기술의 재조합이지 혁신이 아님. MS Recall 대비 방어 논리도 취약
   - 권장 수정 방향: (1) Flow 감지 알고리즘을 핵심 기술로 격상 -- 정확도 벤치마크(기존 타임트래킹 대비), 특허 출원 계획 명시. (2) AI STAR 생성 품질의 정량 평가(hallucination rate, 사용자 수정율, A/B 비교). (3) 시뮬레이션이 아닌 실제 사용 결과물로 교체. (4) 기술 로드맵에 난이도/의존성 분석 추가

3. **[시장성] SOM 달성 경로의 논리적 비약**: 18개월 ARR 5,900만원에서 36개월 SOM 3억원까지 5배 이상 점프하는 경로가 설명되지 않음. Phase 4~5의 "채용 플랫폼 파트너십, B2B 팀 플랜"은 구체적 LoI나 파이프라인 없이 위시리스트에 불과. 또한 유료 전환율 7%->10% 상승의 근거가 "기록이 쌓이면 자연 상승"이라는 희망적 관측
   - 권장 수정 방향: (1) 18개월->36개월 구간의 분기별 MAU/전환율/ARR 성장 경로 구체화. (2) B2B 진입 시 초기 파일럿 기업 1~2곳 확보 또는 LoI. (3) 유료 전환율 근거로 유사 SaaS(Notion, Toggl 등)의 실제 전환율 벤치마크 인용

### 추가 개선 권장사항

4. **[문서 완성도] 액션 아이템 미완 상태 노출**: 심사용 문서 상단에 "미완" 항목이 3개 보이는 것은 "아직 준비 안 됐다"는 신호. 제출 시 반드시 삭제하거나 완료 후 제출

5. **[수치 일관성] TAM 단가 기준 불일치**: 참고자료(메모리)에서 TAM을 연 119,880원(Premium 기준)으로 계산한 적 있으나, 본 문서에서는 연 58,800원(Pro 기준)으로 계산. 일관성 유지 필요. 또한 18개월 MRR 500만원 목표는 실제 계산 시 490만원(1,000명 x 4,900원)으로 미세 불일치 -- 정확한 수치 사용 권장

6. **[리프레이밍 톤] "기술 혁신" vs "습관 설계" 혼선**: 문서 전체적으로 Win32 API, PowerShell 등 기술 용어를 강조하면서 동시에 "귀찮음 제로"라는 UX 가치를 어필. TIPS라면 기술 혁신 쪽으로 일관되게 밀어야 하고, 그렇지 않다면 프로그램 자체를 재고해야 함
